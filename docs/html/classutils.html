int  nrow,

int  step 
)
static

Function to populate the truth table with truth values recursively column wise.

Parameters
arrThe pointer to the truth table which is a 2-D array of integers.
ncolThe number of columns in the truth table.
nrowThe number of rows in the truth table.
columnThe column which is being filled currently.
stepThe number of rows after which the value being filled will be changed. Eg: Step value of 1 means the value will be changed every other row, i.e. 0 1 0 1.... Similarly step of 2 will mean value will change every 2 rows, i.e. 0 0 1 1 0 0 1 1...and so on.
